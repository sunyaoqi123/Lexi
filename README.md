# Lexi - 单词背诵系统

一个功能完整的Android单词背诵应用，包含基础背诵功能和趣味游戏模块。

## 项目结构

```
app/src/main/java/com/syq/lexi/
├── MainActivity.kt                 # 主入口
├── ui/
│   ├── navigation/
│   │   └── MainNavigation.kt       # 导航管理（底部菜单栏）
│   ├── screens/
│   │   ├── HomeScreen.kt           # 首页（圆形背诵按钮 + 背诵计划）
│   │   ├── WordbookScreen.kt       # 单词本选择页面
│   │   ├── GameScreen.kt           # 游戏选择页面
│   │   └── DrawerContent.kt        # 左侧抽屉菜单（我的页面）
│   └── theme/
│       ├── Color.kt                # 颜色配置
│       ├── Theme.kt                # 主题配置
│       └── Type.kt                 # 字体配置
└── data/
    └── model/
        └── Models.kt               # 数据模型定义
```

## 功能模块

### 1. 首页（Home）
- 中央圆形"进入单词背诵"按钮
- 背诵计划展示和管理
- 支持创建、选择、查看进度

### 2. 单词本（Wordbook）
- 显示所有可用的单词本
- 展示单词本信息（总词数、已掌握数、进度）
- 点击进入单词本进行背诵

### 3. 游戏（Game）
- 6种趣味游戏模式：
  - 单词拼写
  - 单词匹配
  - 听音识词
  - 单词接龙
  - 快速反应
  - 单词填空

### 4. 我的（Drawer Menu）
- 用户登录/注册
- 个人资料展示
- 学习统计信息
- 应用设置
- 关于应用

## 技术栈

- **Kotlin** - 编程语言
- **Jetpack Compose** - UI框架
- **Material 3** - 设计系统
- **Android 28+** - 最低API级别

## 快速开始

1. 在Android Studio中打开项目
2. 同步Gradle依赖
3. 运行应用到模拟器或真机
